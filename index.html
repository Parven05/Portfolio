Parven

I love building stuff from scratch, whether it’s a tiny game engine, a graphics tool, or some weird simulation that probably breaks. I mostly use C and C++ because I like having control over how things work. These days, I’m learning Vulkan to really get what’s happening inside the GPU and make things run faster. I also enjoy math and physics, they make it easier, and way more fun, to understand what’s going on behind the scenes.
